HOTS Data Information
The Hawaii Ocean Time-Series (HOTS) program consists of several research components lead by scientists at the University of Hawaii at Manoa. One of these is a physical oceanography component (Roger B. Lukas, PI), which is one of two deep ocean times series hydrographic stations established with support from the US National Science Foundation. (The other time series station is near Bermuda.) The HOTS hydrostation time series is recognized as a WOCE Hydrographic Program Time Series Station.

HOTS Bottle File Types:
.gof:JGOFS
.sea:WOCE
.sum:Summary File
_hy1.csv:WHP Exchange
HOTS CTD File Types:
.sum:Summary File
_ct.zip:WOCE
_ct1.zip:WHP Exchange


For more information and the latest datasets, please go to the HOTS website.
