All files modified by the CCHDO will carry a version number, so that the most recent files may be readily identified. The format for this version number, which will occur at the top of the file (and will not violate any predefined format described in the WOCE 90-1 manual) is:

20080128CCHSIOSCD

where:

20084-digit year of modification
012-digit month
282-digit day
CCHorganization where the modifications took place
SIOinstitution where the organization resides
SCDinitials of the person who modified this file
