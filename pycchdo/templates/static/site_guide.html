Page Conventions

Page Layout

Every page on the website has the CCHDO logo on the top left corner; a search bar at the top right corner; a navigation menu at the top of the page, under the image-rotating banner; and the main content displayed underneath the navigation menu.

The main content is presented in a table or in a modular fashion. Search results are presented in a table; the CCHDO home page and Cruise pages are presented in boxed modules. Additional content may be accessed from links and are presented as blocks of text that appear in a single row beneath the link (Refer to Figure 1).


Figure 1: Page layout with annotated page layout items and examples of additional content accessed by clicking on a link.

Links

With the exception of the navigation menu, links use the following color conventions: link or link in search results, visited link, and hovered link in search results. There is no difference between links with an underline and links without.

Search

There are three (3) ways to conduct a search on the site: 1) Using the Search Bar 2) Navigating through the "BROWSE DATA" menu 3) Using tools from the "SEARCH DATA" menu.

Search Bar

At the top right corner of each page is a search bar where you can search through a keyword/numerical/alphanumeric query.

  • Keyword queries are words associated with an institution, a ship, or the last name of a chief scientist/PI; for example,"NOAA," "Melville," or "Talley."
  • Numerical queries are associated with cruise dates; for example, "2008," "09" (this would display results with a cruise date in the month of September or on the 9th of a month), or "2005-05-20" (this would display results with a cruise date that started or finished on May 20th, 2005).
  • Alphanumeric queries are associated with a WOCE line number or ExpoCode; for example, "A16," "P16," or "316N1981."

You can also search by a particular parameter with the syntax "parameter:query." The following parameters can be used:

    Group
  • Chief_Scientist

For example, "ship:knorr," "line:p10," or "chief_scientist:swift" (See Figure 4). The paramater and query are NOT case-sensitive.

The "Advanced Search" link below the search bar takes you the same page as the Advanced Search page discussed in the "SEARCH DATA" Menu section below.

"SEARCH DATA" Menu

Advanced Search
Instead of searching for data by a keyword, you can refine your search queries by cruises or data files:
  • Search by Cruises - You must set the cruise date range and define at least one of the 5 following parameters in the provided fields of this section to ensure a productive search: (WOCE) Line Number, ExpoCode, Data Contact (Chief Scientist/PI), Ship, and Country. The default value for the cruise date range is from 01-1980 to 01-1980. Once you have defined the refining search parameters, click on the "Find Cruises" button.
  • Search by Data Files - You can refine your search results by indicating the data file type (Sum, Bottle, CTD, Documentation, or all of the above) and the month and year the specified file type(s) were last modified. The default values are "All" file types and a modified date of 01-1994. Once you have defined the refining search parameters, click on the "Find Files" button. The search result page would display modified files from the date you specified until the present.

Interactive Map

To enhance your interactive experience, please make sure you have the most recent version of the Google Earth plugin installed so you can view your search results superimposed on a globe representation of the Earth. On the drop-down menu on the top right corner of the Google Maps that is nominally on "Terrain" mode, choose "Earth" mode to begin using Google Earth capabilities (See Figure &).
